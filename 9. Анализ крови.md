# Анализ крови

## Дано
Дана следующая структура таблиц:

![структура таблиц](https://github.com/Afonina-Olga/ItResumeSQL/blob/main/img/database7.png)

## Задание
Найти все уникальные названия анализов, которые фигурировали в заказах и имеют в названии любую форму слова 'кровь'.

**Примечание.** Обратите внимание, что названия могут быть написаны любым регистром.

## Столбцы в результате
```an_name```

**Важно:** Обратите внимание, что название столбцов в вашем ответе должно в точности совпадать с условием.

## Сортировка
Результат отсортируйте по возрастанию ```an_name```.

## Решение
```  SQL
SELECT DISTINCT an_name
FROM Orders o
INNER JOIN Analysis a ON o.ord_an = an_id
WHERE LOWER(a.an_name) LIKE '%кровь%'
ORDER BY an_name
```
