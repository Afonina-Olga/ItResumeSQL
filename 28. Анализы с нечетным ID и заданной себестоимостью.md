# Анализы с нечетным ID и заданной себестоимостью

## Дано
Дана следующая структура таблиц:

![структура таблиц](https://github.com/Afonina-Olga/ItResumeSQL/blob/main/img/database1.png "Схема БД")

## Задание
Выведите анализы с нечетным ```ID```, у которых себестоимость больше ```100```.

## Столбцы в результате
```an_id``` - ID анализа
```an_name``` - название анализа
```an_cost``` - себестоимость

**Важно:** Обратите внимание, что название столбцов в вашем ответе должно в точности совпадать с условием.

## Решение
``` SQL
SELECT 
  an_id, 
  an_name, 
  an_cost 
FROM 
  Analysis 
WHERE 
  an_cost > 100 
  AND an_id % 2 = 1
```
