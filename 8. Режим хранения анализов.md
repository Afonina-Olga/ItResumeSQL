# Режим хранения анализов

## Дано
Дана следующая структура таблиц:

![структура таблиц](https://github.com/Afonina-Olga/ItResumeSQL/blob/main/img/database7.png)

## Задание
Вывести все заказы, которые содержат анализы, предполагающие режим хранения 22.

## Столбцы в результате
- ```ord_id```
- ```ord_an```
- ```an_name```
- ```gr_temp```

**Важно:** Обратите внимание, что название столбцов в вашем ответе должно в точности совпадать с условием.

## Сортировка
Результат отсортируйте по возрастанию ID заказов.

## Решение
```  SQL
SELECT o.ord_id,
       o.ord_an,
       a.an_name,
       g.gr_temp
FROM Orders o
LEFT JOIN Analysis a ON o.ord_an = a.an_id
LEFT JOIN Groups g ON a.an_group = g.gr_id
WHERE gr_temp = 22
ORDER BY o.ord_id
```
