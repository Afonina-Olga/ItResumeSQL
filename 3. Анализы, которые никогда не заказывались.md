# Анализы, которые никогда не заказывались

## **Дано**

Дана следующая структура таблиц:

![Изображение](https://github.com/Afonina-Olga/ItResumeSQL/blob/main/img/database3.png "структура таблиц")

## **Задание**
Выведите ID и названия всех анализов, которые никогда не заказывались.

## **Столбцы в результате**

- ```an_id``` - ID анализа
- ```an_name``` - название анализа

**Важно:** Обратите внимание, что название столбцов в вашем ответе должно в точности совпадать с условием.

## **Решение:**

``` SQL
SELECT an_id, an_name
FROM Analysis a
LEFT JOIN Orders O
ON a.an_id = o.ord_an
WHERE ord_an IS NULL
```
