# Проранжировать баллы

## Дано

Дана таблица Scores:

```
+----+-------+
| Id | Score |
+----+-------+
| 1  | 3.50  |
| 2  | 3.65  |
| 3  | 4.00  |
| 4  | 3.85  |
| 5  | 4.00  |
| 6  | 3.65  |
+----+-------+
```

## Задание
Нужно проранжировать строки по убыванию поля ```Score```. Если значения совпадают, то нужно присвоить одинаковый ранг. Разрывов в нумерации быть не должно.

**Примечание.** Поле с рангом назовите ```rnk```.

## Столбцы в результате
- ```score``` - номер места студента (автоинкрементное поле, пропусков нет)
- ```rnk``` - ранг

**Важно:** Обратите внимание, что название столбцов в вашем ответе должно в точности совпадать с условием.

## Решение

``` SQL
SELECT 
  Score, 
  DENSE_RANK() OVER (
    ORDER BY 
      Score DESC
  ) AS rnk 
FROM 
  Scores
```
