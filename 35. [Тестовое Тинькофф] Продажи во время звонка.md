# [Тестовое Тинькофф] Продажи во время звонка

## Дано
Дана следующая структура таблиц:

![Схема базы](https://github.com/Afonina-Olga/ItResumeSQL/blob/main/img/database6.png)

## Задание
Получить информацию по продуктам, которые были открыты у клиентов во время звонка с ```05.10.2020``` по текущий день

## Столбцы в результате

- ```last_nm```
- ```first_nm```
- ```middle_nm```
- ```date``` (дата звонка)
- ```open_dt```
- ```product_nm```

**Примечание:** для корректной работы вашего запроса обязательно указывайте схему таблиц - ```tinkoff```. Например, ```FROM tinkoff.employees```.

## Решение
``` SQL
SELECT 
  last_nm, 
  first_nm, 
  middle_nm, 
  start_dttm AS date, 
  open_dt, 
  product_nm 
FROM 
  tinkoff.Customers cu 
  LEFT JOIN tinkoff.Calls ca USING(customer_id) 
WHERE 
  start_dttm >= '2020-10-05' 
  AND open_dt = start_dttm
```
