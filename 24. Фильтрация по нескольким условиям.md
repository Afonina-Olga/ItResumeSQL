# Фильтрация по нескольким условиям

## Задание
В таблице ```Analysis```, найти все уникальные анализы на антитела (содержат ```'anti'```), а также анализы, имеющие в названии любую форму слова ```'кровь'```. В результат вывести только те, у которых розничная цена (```an_price```) меньше 1000.

**Примечание.** Обратите внимание, что названия могут быть написаны любым регистром.

## Столбцы в результате
- ```an_name```
- ```an_price```

**Важно:** Обратите внимание, что название столбцов в вашем ответе должно в точности совпадать с условием.

**Сортировка:** Результат должен быть отсортирован по убыванию розничной цены.

## Решение
``` SQL
SELECT 
  DISTINCT an_name, 
  an_price 
FROM 
  Analysis 
WHERE 
  (
    LOWER(an_name) LIKE '%anti%' 
    OR LOWER(an_name) LIKE '%кровь%'
  ) 
  AND an_price < 1000 
ORDER BY 
  an_price DESC
```
