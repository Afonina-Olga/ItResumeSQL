# Кумулятивное распределение оценок студентов

## Дано
Дана таблица Scores:

```
+----+-------+
| Id | Score |
+----+-------+
| 1  | 3.50  |
| 2  | 3.65  |
| 3  | 4.00  |
| 4  | 3.85  |
| 5  | 4.00  |
| 6  | 3.65  |
+----+-------+
```

## Задание
Требуется вывести долю студентов, у которых оценка меньше или равна текущей оценке в строке.

**Примечания:**

- Поле с долей назовите ```cume_ds```
- Решение должно быть обязательно с использованием функции ```cume_dist```

## Выводимые столбцы
- ```id``` - идентификатор студента
- ```score``` - оценка студента
- ```cume_ds``` - доля студентов, у которых оценка меньше или равна текущей оценке в строке

**Важно:** Обратите внимание, что название столбцов в вашем ответе должно в точности совпадать с условием.

## Решение
``` SQL
SELECT 
  id, 
  score, 
  CUME_DIST() OVER(
    ORDER BY 
      score
  ) AS cume_ds 
FROM 
  Scores
```
