# Наименее заказываемая группа

## Дано
Дана следующая структура таблиц:

![структура таблиц](https://github.com/Afonina-Olga/ItResumeSQL/blob/main/img/database1.png)

## Задание
Найдите группу, анализы которой заказывались меньше всего.

## Столбцы в результате
```gr_name```

```Важно:``` Обратите внимание, что название столбцов в вашем ответе должно в точности совпадать с условием.

## Решение
``` SQL
SELECT 
  gr_name 
FROM 
  Orders o 
  JOIN Analysis a ON o.ord_an = a.an_id 
  JOIN Groups g ON g.gr_id = a.an_group 
GROUP BY 
  gr_name 
ORDER BY 
  COUNT(*) 
LIMIT 
  1
```
