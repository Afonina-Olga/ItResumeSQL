# [Тестовое Тинькофф] Недозвонившиеся клиенты

## **Дано**
Дана следующая структура таблиц:

![Схема базы](https://github.com/Afonina-Olga/ItResumeSQL/blob/main/img/database6.png)

## **Задание** 
Получите информацию по клиентам (таблица ```Customers```), которые звонили и не дозвонились ```29.05.2019```.

**Столбцы в результате**

- date (дата звонка)
- last_nm
- first_nm
- middle_nm

**Примечание:** для корректной работы вашего запроса обязательно указывайте схему таблиц - ```tinkoff```. Например, ```FROM tinkoff.employees```.

## Решение
```
SELECT ca.start_dttm AS date,
       cu.last_nm,
       cu.first_nm,
       cu.middle_nm
FROM tinkoff.Customers cu
INNER JOIN tinkoff.Calls ca ON ca.customer_id = cu.customer_id
WHERE ca.dozv_flg = 0
  AND ca.start_dttm = '2019-05-29'
```
