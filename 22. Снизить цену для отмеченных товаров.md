# Снизить цену для отмеченных товаров

## Дано
Дана следующая структура таблиц:

![структура таблиц](https://github.com/Afonina-Olga/ItResumeSQL/blob/main/img/database20.png)

## Задание
Написать ```SELECT```-запрос для уменьшения цены на ```20%``` для тех продуктов в таблице ```STORE```, которые содержат пометку ```note = DISCOUNT```.

**Примечание.** Если пометки нет, то необходимо оставить исходную цену.

## Столбцы в результате
- всё из таблицы ```STORE```
- ```price_with_discount``` - рассчитанная цена

**Важно:** Обратите внимание, что название столбцов в вашем ответе должно в точности совпадать с условием.

## Дополнительные условия
**Важно:** Обратите внимание, что Вам нужно написать именно ```SELECT```-запрос, а не ```UPDATE```, т.е. содержимое таблицы в БД менять не нужно.

## Решение
``` SQL
SELECT 
  *, 
  CASE WHEN note = 'DISCOUNT' THEN unit_price * 0.8 ELSE unit_price END AS price_with_discount 
FROM 
  store
```
