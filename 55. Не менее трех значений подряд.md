# Не менее трех значений подряд

## Дано
Дана таблица Logs:

```
+--------+---------+
|   ID   |   Num   |
+--------+---------+
```

## Задание
Найти все ```Num```, которые встречаются не менее трех раз подряд.

## Столбцы в результате
```ConsecutiveNums``` - результат

**Примечание.** Числа выведите без повторов - чтобы одно число в столбце ```ConsecutiveNums``` встречалось только ```1``` раз.

**Важно:** Обратите внимание, что название столбцов в вашем ответе должно в точности совпадать с условием.

## Сортировка
Результат необходимо отсортировать по убыванию.

## Решение
``` SQL
SELECT 
  Num AS ConsecutiveNums 
FROM 
  Logs 
GROUP BY 
  Num 
HAVING 
  COUNT(*) > 2 
ORDER BY 
  ConsecutiveNums DESC
```
