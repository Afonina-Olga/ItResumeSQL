# [Тестовое Тинькофф] Отчет о зарплате

Дана следующая структура таблиц:

![Схема базы](https://github.com/Afonina-Olga/ItResumeSQL/blob/main/img/database6.png)

## Задание
Компании необходимо создать отчет, в котором будут перечислены все сотрудники, которые были наняты в течение определенных дат и чья зарплата попадает в некоторый диапазон. Этот отчет будет использоваться финансовым отделом компании для анализа расходов компании на заработную плату и выявления любых тенденций или отклонений.

Напишите запрос, который на дату ```'2022-10-10'``` выведет всех актуальных сотрудников, имеющих заработную плату в пределах от ```20000``` до ```50000```.

## Столбцы для вывода

- ```last_nm```
- ```first_nm```
- ```middle_nm```
- ```salary```
- ```hire_dt```

**Примечание:** для корректной работы вашего запроса обязательно указывайте схему таблиц - ```tinkoff```. Например, ```FROM tinkoff.employees```.

## Решение
``` SQL
SELECT 
  last_nm, 
  first_nm, 
  middle_nm, 
  salary, 
  hire_dt 
FROM 
  tinkoff.Employees e 
  LEFT JOIN tinkoff.Job j USING(position_id) 
WHERE 
  (
    j.valid_to_dttm IS NULL 
    OR j.valid_to_dttm >= '2022-10-10'
  ) 
  AND j.salary BETWEEN 20000 AND 50000
```
