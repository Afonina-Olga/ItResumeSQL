# [Тестовое Тинькофф] DAU

Дана следующая структура таблиц:

![Схема базы](https://github.com/Afonina-Olga/ItResumeSQL/blob/main/img/database6.png)

## Задание
Давайте найдем условное ```DAU``` по дате открытия продукта у клиента. Посмотрим, сколько в среднем уникальных клиентов открывают у нас продукт каждый день.

## Столбцы в результате

```DAU``` (одно значение, округленное до 2х знаков после запятой)

**Примечание:** для корректной работы вашего запроса обязательно указывайте схему таблиц - ```tinkoff```. Например, ```FROM tinkoff.employees```.

## Решение
``` SQL
WITH cte AS (
  SELECT 
    COUNT(customer_id) AS DAU 
  FROM 
    tinkoff.Customers 
  GROUP BY 
    open_dt
) 
SELECT 
  ROUND(
    AVG(DAU), 
    2
  ) 
FROM 
  cte
```
