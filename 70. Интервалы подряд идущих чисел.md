# Интервалы подряд идущих чисел

## Дано
Дана таблица ```Numbers```, которая представляет из себя последовательность чисел:

```
+-----+
|  a  |
+-----+
|  1  |
|  2  |
|  3  |
|  4  |
|  6  |
|  7  |
|  10 |
|  11 |
|  12 |
|  13 |
|  15 |
|  17 |
+-----+
```

## Задание
Задача состоит в том, чтобы вернуть интервалы подряд идущих чисел.

## Столбцы в результате
- ```start``` - начало интервала
- ```end``` - конец интервала

Результатом будет таблица:

```
+-------+-----+
| start | end |
+-------+-----+
|   1   |  4  |
+-------+-----+
|   6   |  7  |
+-------+-----+
|  10   | 13  |
+-------+-----+
|  15   | 15  |
+-------+-----+
|  17   | 17  |
+-------+-----+
```

**Важно:** Обратите внимание, что название столбцов в вашем ответе должно в точности совпадать с условием.

## Сортировка
Результат отсортируйте по возрастанию интервалов.

## Решение
``` SQL
WITH a AS (
  SELECT 
    a, 
    a - ROW_NUMBER() OVER (
      ORDER BY 
        a ASC
    ) AS diff 
  FROM 
    Numbers
) 
SELECT 
  MIN(a) AS start, 
  MAX(a) AS end 
FROM 
  a 
GROUP BY 
  diff 
ORDER BY 
  start
```
