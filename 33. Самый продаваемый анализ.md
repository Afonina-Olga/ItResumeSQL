# Самый продаваемый анализ

## Дано
Дана следующая структура таблиц:

![структура таблиц](https://github.com/Afonina-Olga/ItResumeSQL/blob/main/img/database1.png "Схема БД")

## Задание
Найти анализ, который продавался больше всего раз за весь период времени.

## Столбцы в результате
- ```an_id```
- ```cnt``` - рассчитанное количество

**Важно:** Обратите внимание, что название столбцов в вашем ответе должно в точности совпадать с условием.

## Решение
``` SQL
SELECT 
    ord_an AS an_id, 
    COUNT(ord_an) cnt 
  FROM 
    Orders 
  GROUP BY 
    ord_an
  ORDER BY cnt DESC
  LIMIT 1
```
